## Getting Started

### Prerequisites
- **Node.js** 18+
- **Python** 3.8+
- **AstraDB account**
- Basic understanding of:
  - React/Next.js
  - 3D graphics
  - Database operations

### Installation Steps
1. Clone repository:
   ```bash
   git clone https://github.com/your-username/soulbuddy.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables.
4. Set up database connections.
5. Run development server:
   ```bash
   npm run dev
   ```
